# Changelog

All notable changes to Silent Signal will be documented in this file.

## [1.0.0] - 2025-12-01

### Added
- Initial MVP release
- Real-time gesture detection via WebSocket
- AI-powered intention classification with Cerebras
- Text-to-speech generation with ElevenLabs
- User authentication with WorkOS
- Payment processing with Stripe
- Search and analytics with Searchable
- SmartFlow orchestration with Raindrop
- Complete frontend with Jinja2 templates
- Comprehensive API documentation
- Docker deployment configuration
- Vultr deployment scripts
- Cloudflare integration guide

### Integrations
- WorkOS - Authentication
- Cerebras - AI Inference
- ElevenLabs - Text-to-Speech
- Raindrop (LiquidMetal AI) - Orchestration
- Stripe - Payments
- Searchable - Search & Analytics
- Vultr - Hosting
- Cloudflare - CDN & Security

### Features
- Blink detection
- Tap gesture recognition
- Micro-gesture support
- Real-time audio generation
- Gesture history tracking
- User preferences management
- Premium subscriptions
- Search functionality
- Analytics dashboard
- Rate limiting
- Caching with Redis
- Structured logging
